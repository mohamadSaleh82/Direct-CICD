# Direct-Deploy (CI/CD Tool)

این ابزار برای همگام‌سازی لحظه‌ای (Real-time Sync) بین سیستم شما و سرور از طریق SSH طراحی شده است.

## نحوه استفاده

1. **نصب پیش‌نیازها:**
   در صورتی که کتابخانه‌های لازم نصب نشده‌اند، دستور زیر را در ترمینال اجرا کنید:
   ```bash
   pip install -r requirements.txt
   ```

2. **اجرای ابزار:**
   برای شروع مانیتورینگ و انتقال تغییرات، دستور زیر را اجرا کنید:
   ```bash
   python direct_deploy.py
   ```

## تنظیمات (config.json)

شما می‌توانید تنظیمات سرور و مسیرها را در فایل `config.json` تغییر دهید:
- `server`: اطلاعات دسترسی به سرور (Host, Username, Password).
- `paths`: مسیر محلی (Local) و مسیر روی سرور (Remote).
- `ignore`: فایل‌ها یا پوشه‌هایی که نمی‌خواهید منتقل شوند (مثل `.git` یا `node_modules`).

## ویژگی‌ها
- **Live Sync:** به محض ذخیره فایل در سیستم خودتان، تغییرات به سرور منتقل می‌شود.
- **Auto Reconnect:** در صورت قطع شدن اینترنت یا SSH، ابزار تلاش می‌کند مجدداً وصل شود.
- **Efficient:** فقط فایل‌هایی که تغییر کرده‌اند منتقل می‌شوند.
